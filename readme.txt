＊紹介

HTML では、 img 要素の alt 属性（画像の代替テキスト）に、画像を表示しな
い環境で画像の代わりに表示するためのテキストを指定しなければなりません。

この alt 属性の値は、 IE や NN4 などでは ポップアップ表示されるのですが、
Mozilla/Netscape 6/7 ではポップアップ表示されません。（そのため、 alt を
「ポップアップ表示用のテキスト」だと勘違いして使っているせいで alt がポッ
プアップ表示されないと何がなんだかわからなくなってしまっているサイトとい
うのが多数存在します。）このパッケージを導入すると、 Navigator ウィンドウ
において、 alt 属性の内容をポップアップ表示するようになります。

おまけとして、ツールチップの内容が長い場合に、内容を折り返し表示する機能
があります。



以下、隠し機能の使い方。

user_pref("browser.chrome.tooltips.attrlist.enabled", true);
user_pref("browser.chrome.tooltips.attrlist",
  "alt|src|data|title|href|cite|action|onclick|onmouseover|onsubmit");

という風な行をuser.jsに書き加えるか、about:configでこれらの設定を行うと、
リストに記述された属性全てをポップアップ表示するようになります。
また、ポイントした要素がその属性を持っていなくても、その属性を持った要素
が祖先にある場合、それらも表示します。

user_pref("browser.chrome.tooltips.attrlist.recursively", true);

を設定すると、全ての祖先要素について属性値を表示します。
これらの機能を使うと、もしかしたらフィッシング詐欺の予防になるかもしれま
せん。

========================================================================
＊アンインストールの手順

このパッケージには自動で自身をアンインストールする機能はありません。
以下の手順に従って、手動でパッケージを削除して下さい。

  1. NS/Moz を終了させる。簡易起動（高速起動）が有効になっている場合、
     タスクトレイのアイコンを右クリックして、そちらも終了させる。
  2. ユーザープロファイルディレクトリか、 NS/Moz をインストールした
     ディレクトリ（管理者権限でインストールした場合）の中にある
     /chrome/ フォルダから、 popupalt.jar, overlayinfo ディレクトリ,
     chrome.rdf （ NS6 の場合は all-packages.rdf ）を削除する。
  3. 管理者権限でインストールした場合、installed-chrome.txt をメモ帳等
     で開き、以下の行を削除する。
     ・content,install,url,jar:resource:/chrome/popupalt.jar!
                           /content/popupalt/
